# Portuguese Missa Terminology Glossary

Project glossary for consistent review/correction of Portuguese liturgical text.

## Section-level Terms

- `Introitus` -> `Intróito`
- `Oratio` -> `Oração` / `Coleta` (keep corpus-consistent per source tradition)
- `Lectio` -> `Lição` / `Epístola` (context-dependent)
- `Graduale` -> `Gradual`
- `Tractus` -> `Trato`
- `Evangelium` -> `Evangelho`
- `Offertorium` -> `Ofertório`
- `Secreta` -> `Secreta`
- `Communio` -> `Comunhão`
- `Postcommunio` -> `Pós-comunhão`

## Recurring Liturgical Formulae

- `The Lord be with you.` -> `O Senhor esteja convosco.`
- `And with your spirit.` -> `E com o teu/vosso espírito.` (choose one register and keep consistent)
- `Let us pray.` -> `Oremos.`
- `Through our Lord Jesus Christ...` -> `Por Nosso Senhor Jesus Cristo...`
- `Amen.` -> `Amém.`

## Consistency Rules

- Keep structural markers untouched: `@`, `!`, `&`, `$`, and section brackets.
- Keep biblical citations and references unchanged.
- Preserve line order and section order for parser compatibility.
- Prefer Catholic liturgical Portuguese register; avoid colloquial wording.
