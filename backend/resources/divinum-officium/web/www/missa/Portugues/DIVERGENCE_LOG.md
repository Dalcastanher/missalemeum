# Portuguese Missa Divergence Log

Tracks differences discovered during English-to-Portuguese preparation and external comparison.

## Current Baseline

- Imported official Portuguese corpus where available.
- Created Portuguese-side counterparts for all English filenames to unblock comparison workflow.
- Remaining task is manual correction/reconciliation for gap-filled files.

## Known Divergence Categories

1. **Coverage divergence**
   - Portuguese source contains some files not present in English.
   - English source contains files that had no official Portuguese counterpart and were gap-filled.

2. **Wording divergence**
   - Official Portuguese source may use different liturgical phrasing than modern PT-BR references.
   - Gap-filled files need manual revision to avoid English leftovers.

3. **Rubrical/register divergence**
   - Formal pronoun usage (`tu` vs `vós`) and capitalization conventions may differ by source.
